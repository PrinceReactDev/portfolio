UX/ UI Designer

UX/ UI Designer :

Job Responsibilities:

  • Create and manage core experiential design concepts as per project requirements.
  • Turn visions into concepts, and translate those concepts into recommendations that achieve simplicity,
  • Produce best in class UX/UI deliverables including wire frames, storyboards, user flows, prototypes, process flows, logo designs, app icons, etc
  • Develop visual designs for websites, e mails, Brochure and mobile applications.
  • Research and track advancements in web and mobile application design patterns.

Skills:

  • Solid experience in creating wire frames, storyboards, user flows, process flows and site maps.
  • Proficiency in Photoshop, Illustrator or other visual design and wire-framing tools.
  • Strong Communication & Presentation skills with the ability to listen to, and build rapport with clients.
  • Knowledge of HTML, CSS and JavaScript frameworks Good at teamwork.
  • Should be fluent in English.
  • Proven ability to deliver high quality designs to customers.
  • Good understanding of Google’s material design guidelines and Apple’s human interface design guidelines.
